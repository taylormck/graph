bool Graph::valid ( ) const [inline, private]

Checks the preconditions of the graph Preconditions currently checked

  • Every vertex has the identity element as the first element
  • There are no parallel paths
  • The member numEdges correctly represents the number of edges in the graph
Returns:
true if the preconditions are met false otherwise
